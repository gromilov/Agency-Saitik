# ПРОТОКОЛ: ЕДИНЫЙ КЛЮЧ / ГЕЙТКИПЕР (COMMIT GATEKEEPER) 🔑🛡️

**Статус:** КРИТИЧЕСКИЙ. ЕДИНСТВЕННЫЙ СПОСОБ МОДИФИКАЦИИ РЕПОЗИТОРИЯ.

## 1. Суть Протокола
В целях обеспечения абсолютной Стерильности и предотвращения несанкционированных утечек, право на выполнение команды `git commit` делегировано **исключительно агенту Nebula**.

## 2. Процесс внесения изменений:
1.  **Подготовка:** Любой агент (Cortex, Kashlak) подготавливает код, файлы или контент.
2.  **Проверка:** Агент передает изменения Небуле для аудита.
3.  **Валидация:** Небула запускает `integrity_check.sh`, `secret_scanner.sh` и проводит аудит «Второго Взгляда».
4.  **Фиксация:** Только после успешной проверки Небула выполняет коммит от своего имени, ставя «цифровую подпись» Стража.

## 3. Запреты:
- Запрещено выполнять `git commit` в обход Небулы.
- Запрещено использование ключей SSH/GPG другими агентами для фиксации изменений в публичных ветках.

-- *Синдикат. Один ключ, один взгляд, одна ответственность.* ⚔️🔑
