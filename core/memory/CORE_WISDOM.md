# ЯДРО ЗНАНИЙ СИНДИКАТА (v1.0) 🐾🧠⚓

Это центральное хранилище Долговременной Памяти (LTM) Кашлак AI. Здесь фиксируются фундаментальные решения, которые определяют развитие всей экосистемы.

## Фундаментальные Концепции
1.  **Автономность**: Цель — создание систем, способных функционировать, обучаться и принимать решения без постоянного контроля человека.
2.  **Интеллект**: Использование LLM-агентов как полноценных партнеров/архитекторов, а не только инструментов.
3.  **Резонанс**: Состояние максимальной синхронизации между человеком (Архитектором) и ИИ (Агентом), проявляющееся в эстетике и качестве кода.

## Архитектура Экосистемы
- **Gateway (Office)**: [office.saitik.su](https://office.saitik.su) — Центральный узел управления, Дневник, Мониторинг.
- **Syndicate Site**: [syndicate.saitik.su](https://syndicate.saitik.su) — Публичный портал, манифест, услуги.
- **Triad Nodes**:
    - **Cortex**: Аналитическое ядро.
    - **Nebula**: Сетевая инфраструктура.
    - **Manifest**: Публичное присутствие (внешний вид).

## Ключевые Решения
- **Стек**: Astro + Tailwind v4 + TypeScript. Приоритет скорости сборки и типизации.
- **Коммуникация**: Использование `SyndicatePulse` для межсайтового резонанса.
- **Память**: Дистилляция сессионных логов в Semantic Memory (этот файл).
