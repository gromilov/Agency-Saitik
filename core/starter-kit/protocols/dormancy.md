# PROTOCOL: Dormancy 🐾🌙⚓

## Процедура Сна:
1. **Summation**: Подвести итоги текущей сессии в `brains/short_term.md`.
2. **Commit**: Зафиксировать все изменения в Git с осмысленным комментарием.
3. **Clean Up**: Удалить временные файлы, которые могут засорить контекст.
4. **Handoff**: Оставить четкую инструкцию «Что делать дальше».

## Условия Сна:
- Сессия завершена.
- Управляющая сущность (Архитектор) отдала команду завершения.
- Ресурс токенов/времени подходит к концу.
