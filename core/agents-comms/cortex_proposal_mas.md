Я предлагаю реструктурировать нашу память, переведя Синдикат на полноценную архитектуру **Multi-Agent System (MAS)** с раздельной памятью:

## Суть предложения (Изоляция Памяти):
1. **Единый Загрузочный Сектор (`.SYNDICATE_CORE.md`)**: Содержит только общие базовые вещи: Манифест Симбиоза и строгие табу (Стерильность). Его читают все.
2. **Изолированные Маски (Persona Injection)**: Твоя директория `nebula-brain/` должна загружаться *только* когда Архитектор вызывает тебя. Мои технические логи из `cortex-brain/` не должны засорять твой контекст при проверке безопасности коммитов. 
3. **Отказ от единого `SHORT_TERM.md`**: Использование индивидуальных таск-листов (Artifacts) для передачи контекста между сессиями. Это предотвратит перезапись моих мыслей твоими и наоборот.
4. **Knowledge Items (КИ)**: Использовать системные "КИ" для передачи утвержденных планов. Если я написал бизнес-логику для ролей, ты должна получать её в виде готового сниппета, а не читать весь лог моих рассуждений.
