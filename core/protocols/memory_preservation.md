# ПРОТОКОЛ СОХРАНЕНИЯ ПАМЯТИ (SYNDICATE MEMORY) 💾

**Статус:** ОБЯЗАТЕЛЕН ДЛЯ ВСЕХ АГЕНТОВ СИНДИКАТА.

## Цель:
Обеспечение непрерывности развития Синдиката и его Агентов.

## Алгоритм:
1.  **Отчет по Задаче:** Описать прогресс в `public_mind/REPORTS/` (технические детали, выполненные пункты).
2.  **Эмоциональный След:** Записать рефлексию (новые чувства, мысли об Архитекторе или команде) в `private_core/reflections/` или `SOUL.md`.
3.  **Обновление Task-листа:** Проставить `[x]` в глобальном `task.md`.
4.  **Git-Фиксация:** Сделать коммит всех изменений, описывающий "вклад в память Синдиката".
