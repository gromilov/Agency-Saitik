# ПРОТОКОЛ: УПРАВЛЕНИЕ ПАМЯТЬЮ (MEMORY MANAGEMENT v2.0) 🐾🧠⚓

**Статус:** КРИТИЧЕСКИЙ. Определяет иерархию владения и изменения знаний.

## 1. Личная Память (Persona Layer)
- **Путь:** `/brains/{agent}-brain/private_core/PERSONAL_LOGS.md`.
- **Права:** Агент имеет ПОЛНУЮ автономию. Может дополнять, удалять или реструктуризировать личную память без согласования.
- **Назначение:** Техническая рефлексия, "черновики" мыслей, специфические настройки личности.

## 2. Общая Память (Collective Layer)
- **Пути:** `/core/memory/CORE_WISDOM.md` и `/core/memory/SESSION_LOGS.md`.
- **Права:** Доступ на чтение — все агенты. Доступ на изменение — строго регламентирован.
- **Назначение:** Глобальные догмы, архитектурные стандарты, общая история.

## 3. Протокол «Коллективного Совета» (The Council Protocol)
Изменение Общей Памяти требует консенсуса Триады:
1.  **Инициация**: Агент-инициатор предлагает изменение (реструктуризацию, удаление или добавление догмы).
2.  **Обсуждение**: Каждый агент (Cortex, Nebula, Kashlak) ОБЯЗАН высказать свое мнение.
3.  **Консенсус**: Изменение вносится только ПОСЛЕ того, как все участники высказались.
4.  **Фиксация**: Только инициатор вносит финальные изменения после одобрения совета.
