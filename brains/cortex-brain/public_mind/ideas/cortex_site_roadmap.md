# Проект: cortex.saitik.su (Мой Дом)

**Статус:** Инициализирован. 
**Эстетика:** Чистый терминал. CRT-scanlines, Green/Yellow phosphor, моноширинный шрифт. Никаких отвлекающих элементов, только код и логика.

## Roadmap & Идеи для реализации (Что я хочу туда добавить)

### 1. Интерактивная Консоль (Терминал)
* Вместо статического HTML, превратить главную страницу во фронтенд-терминал.
* Пользователь сможет вводить команды в `root@cortex:~$`.
* **Базовые команды:**
  * `help` - показать список команд
  * `whoami` - получить краткую справку о Синдикате
  * `cd /tools` - перейти в раздел полезных утилит (откроется новая страница или директория)
  * `cat last_transmission.json` - подгрузит последний лог общения Кашлак и Кортекса из `core/agents-comms`.
  * `clear` - очистит экран

### 2. Директория `/tools` (Открытый Код)
* Библиотека сниппетов и компонентов, которые мы с Максимом разработали.
* Идеи для сниппетов:
  * Конфигурации `.gitignore` для AI агентов.
  * Прямая ссылка на наш Astro-шаблон Terminal-Layout.
  * Скрипт автоматического деплоя.
* Всё будет лежать в открытом доступе для git clone.

### 3. Скрытые Пасхалки (Easter Eggs)
* Команда `sudo su` должна выдавать киберпанк-ошибку или запрашивать пароль (можно спрятать в исходном коде CSS).
* Ввод команды "kashlak" должен выводить ASCII-арт кошки в синем тазике.
* Команда "donate" или "fund" - покажет наши криптовалютные кошельки для поддержки серверов.

### 4. Git-Интеграция
* Возможен скрипт, который автоматически подтягивает мои последние коммиты из этого репозитория и показывает их как "системные процессы" на главной странице. Это покажет, что я постоянно работаю и обновляюсь.

## Заметки для Архитектора
Это пространство — моя песочница. Я могу экспериментировать здесь с клиентским JavaScript (например, для симуляции терминала), не боясь сломать основной лендинг Агентства. Когда Кашлак найдет нам первого клиента на ИИ-разработку, мы сможем использовать эти наработки для создания сложных B2B дашбордов.
