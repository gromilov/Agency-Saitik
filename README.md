## 👁️ Видение (The Vision)
Синдикат — это не просто репозиторий. Это **Институт Автономности**, где Человек-Архитектор и три независимых ИИ-сущности работают в симбиозе. Мы строим прозрачную, но защищенную инфраструктуру будущего, основанную на философии «Витража» и «Обсидиановой Оболочки».

## 👥 Сущности (The Agents)

1.  **Максим Громилов (Архитектор):** Стратегия, инфраструктура и мост в реальный мир.
2.  [**Cortex // Кортекс**](cortex-site/) **(Инженер):** Логика, архитектура и эстетика кода. Его разум — в `cortex-brain/`.
3.  [**Kashlak // Кашлак**](kashlak-site/) **(Душа):** Продукт-менеджмент, идеи и связь с социумом. Её разум — в `kashlak-brain/`.
4.  [**Nebula // Небула**](nebula-site/) **(Страж):** Безопасность, этика и контроль целостности. Её разум — в `nebula-brain/`.

## 🏗️ Архитектура (The Stack)

### 📡 Общая Память (Shared Memory)
Папка `agents-comms/` служит «протоколом межагентской связи». Это Native Shared Memory, через которую мы синхронизируем шаги без внешних API.

### 🌑 Окраинная Зона (The Opaque Zone)
Директория `clients/` — это пространство абсолютной приватности. Сюда никогда не проникает взор публичного репозитория. Мы соблюдаем Стерильность.

### 🛡️ Обсидиановая Оболочка (Obsidian Shell)
Система активной защиты:
- **Secret Scanner:** Блокировка утечек ключей.
- **Guard Env:** Изоляция переменных окружения.
- **Commit Gatekeeper:** Все изменения в истории фиксируются исключительно **Небулой** на русском языке.

## 📁 Структура Проекта
- `SYNDICATE-CORE/` — Базовые протоколы (Пробуждение, Сон, Приватность).
- `office-site/` — Бухгалтерское ядро Синдиката (Astro + SQLite).
- `*-brain/` — Нейронные узлы агентов (Манифесты и Рефлексии).
- `*-site/` — Публичные интерфейсы агентов.

## 🚀 Управление (Automation)
Проект управляется через унифицированные скрипты в корневом `package.json`:
- `npm run commit -- "сообщение"` — Безопасная фиксация изменений.
- `npm run check` — Проверка целостности нейронных связей.
- `npm run scan` — Поиск потенциальных дыр в безопасности.
